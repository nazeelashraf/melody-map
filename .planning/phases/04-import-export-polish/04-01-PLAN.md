Create two new components following the project's inline React.CSSProperties pattern:

**ExportButton.tsx:**
- Props: `data: Sheet | Composition`, `label?: string` (default "Export")
- On click: serialize `data` to JSON with `JSON.stringify(data, null, 2)`, create a `Blob` with type `application/json`, create `URL.createObjectURL(blob)`, create an invisible `<a>` element, set `a.href = url`, set `a.download = "{data.title.replace(/[^a-zA-Z0-9]/g, '-').toLowerCase()}.json"`, trigger `a.click()`, then call `URL.revokeObjectURL(url)` after a 100ms timeout.
- Style: small pill-shaped button matching the existing `secondaryButtonStyle` pattern (border: 1px solid #cbd5e1, borderRadius: 999px, padding: 0.45rem 0.75rem, backgroundColor: #fff, color: #334155, fontSize: 0.8rem, fontWeight: 600, cursor: pointer).
- Use a download icon character ⬇ before the label text.

**ImportDialog.tsx:**
- Props: `mode: 'sheet' | 'composition'`, `onClose: () => void`
- Internally uses `useSheet()` and `useComposition()` to get dispatch.
- Renders a modal overlay (fixed position, semi-transparent backdrop) with a centered card containing:
  - Title: "Import {mode}"
  - A hidden `<input type="file" accept=".json">` triggered by a visible "Choose File" button
  - An error display area (red text, hidden when no error)
  - A "Cancel" button that calls `onClose()`
- On file selection:
  1. Read file with `FileReader.readAsText(file)`
  2. In `onload`: `JSON.parse(event.target.result)` wrapped in try/catch
  3. If parse fails: show error "Invalid JSON file: {parse error message}"
  4. If parse succeeds: call `sheetSchema.safeParse(parsed)` or `compositionSchema.safeParse(parsed)` depending on mode
  5. If Zod validation fails: show error "Invalid data: {first error message from Zod's error.issues[0].message}"
  6. If Zod validation succeeds: assign a NEW `crypto.randomUUID()` as the id (replace `result.data.id` with a fresh UUID), then dispatch `CREATE_SHEET` or `CREATE_COMPOSITION` with the validated+re-IDed data, then call `onClose()`
  7. In `onerror`: show error "Failed to read file"
- Style: modal overlay uses `position: fixed, inset: 0, backgroundColor: rgba(0,0,0,0.4), display: grid, placeItems: center, zIndex: 50`. Card uses `backgroundColor: #fff, borderRadius: 18px, padding: 1.5rem, maxWidth: 28rem, width: 90%`. Error text uses `color: #dc2626, fontSize: 0.875rem, marginTop: 0.75rem`.
