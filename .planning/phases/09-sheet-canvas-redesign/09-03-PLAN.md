<objective>
Redesign the instrument context controls to use a track-style visual language inspired by DAW track headers. Apply colored track headers for the global instrument toggle in the editor and update PerformanceView tabs to match. Ensure per-line instrument overrides also use track-style selection.

Purpose: Musicians working with multiple instruments need immediate visual feedback about which instrument context is active. Track-style headers with stable instrument identity colors (piano=blue, guitar=green, drums=purple) provide this clarity.
Output: SheetEditor with track-style instrument headers and PerformanceView with track-style instrument tabs.
</objective>
