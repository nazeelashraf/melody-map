Redesign each lyric line's editing block so cues and lyrics are visually distinct lanes on the canvas.

Requirements per CANV-02:
1. **Remove heavy article container**:
   - Replace `rounded-xl border bg-muted/40 p-4 space-y-3` with a lighter structure
   - Use `py-3 border-b border-canvas-muted` as the container — no background, no rounded corners, just a subtle bottom border separating lines
   - Last line should not have a bottom border (or use `divide-y` on the parent)

2. **Cue lane styling**:
   - The cue input row should have a subtle cue-specific background to distinguish it from lyrics
   - Piano cues: `bg-piano-muted` (or `bg-blue-500/5` if token not available) with a very subtle left border `border-l-2 border-piano`
   - Guitar cues: `bg-guitar-muted` with `border-l-2 border-guitar`
   - Drum cues: `bg-drums-muted` with `border-l-2 border-drums`
   - The CueRowInput itself should blend into this lane — use `bg-transparent border-0 focus-visible:ring-1 focus-visible:ring-accent` for the input
   - Remove the rounded-xl and heavy border from the cue input

3. **Lyric lane styling**:
   - Lyric input should have NO special background — it sits on the canvas background
   - Use `bg-transparent border-0 border-b border-border/50 focus-visible:border-accent focus-visible:ring-0` for a clean underline style
   - This makes lyrics feel like the "ground" and cues feel like "layers" above

4. **Lane labels**:
   - Move labels inside the lane or remove them entirely — the instrument color + position is enough context
   - If kept, use `text-[10px] uppercase tracking-wider text-muted-foreground` placed above the lane
   - For lyrics, no label needed

5. **Guide markers**:
   - Keep GuideMarker functionality exactly as-is
   - Adjust `toneClassName` to match the new lane styling:
     - Cue guide: `bg-accent/20` (subtle)
     - Lyric guide: `bg-foreground/10` (subtle)

6. **Vertical rhythm**:
   - Cue lane and lyric lane should be tightly spaced (`space-y-0` or `gap-0`) to emphasize their vertical linkage
   - Use `grid gap-0` or `flex flex-col gap-0` for the lane container
