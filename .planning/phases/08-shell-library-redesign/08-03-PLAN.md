Redesign the SheetCard to communicate sheet identity more clearly and provide better visual hierarchy.

Requirements per LIB-01 and UI-01:
1. **Card container**:
   - Use `bg-card` background with `border border-border`
   - Increase padding (`p-5`)
   - Use subtle shadow (`shadow-sm`) that increases on hover (`hover:shadow-md`)
   - Add transition for shadow and border-color (`transition-all duration-200`)
   - On hover, border should subtly darken (`hover:border-border/80`)

2. **Title area**:
   - Title should use `text-lg font-semibold text-foreground`
   - Inline rename Input should match title styling (`text-lg font-semibold`)
   - Add a music note icon (Music from lucide-react) next to title in accent color
   - Rename pencil icon should appear on hover (`group-hover:opacity-100`) with transition

3. **Metadata row**:
   - Display tempo with Clock icon: `{tempo} BPM`
   - Display lyric line count with TextLines icon (or AlignLeft): `{count} lines`
   - Use `text-sm text-muted-foreground` for metadata
   - Add small dot separator (•) between metadata items

4. **Instrument identity hints**:
   - Add a subtle row of instrument indicators at the bottom of the card
   - Show small colored dots or labels for Piano (blue), Guitar (green), Drums (purple)
   - Only show instruments that have arrangement content (non-empty)
   - Use instrument color tokens: `text-piano`, `text-guitar`, `text-drums`
   - This provides immediate visual scanning for instrument coverage

5. **Actions**:
   - Open link should be more prominent — use a button-like style with `bg-accent text-accent-foreground hover:bg-accent-hover`
   - Or keep the link style but make it clearer it's the primary action
   - Delete button should remain subtle (`variant="ghost" size="icon"`) but use `hover:text-destructive hover:bg-destructive/10`
   - Actions should be at the bottom of the card, right-aligned

6. **Focus and selection**:
   - Card should have `focus-visible:ring-2 focus-visible:ring-focus` when focused
   - Ensure the card is not a focus trap — only interactive elements receive focus

7. **Responsive**:
   - Card should not overflow its grid cell
   - Text should truncate (`truncate` or `line-clamp-1`)
